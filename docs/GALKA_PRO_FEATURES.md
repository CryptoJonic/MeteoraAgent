# Galka Pro feature scope

Galka Pro is a custom TradingView-like terminal built on the open-source Lightweight Charts library. It does not copy TradingView proprietary source code, private libraries, trademarks, or hosted data.

## Implemented

- Binance USD-M Futures live market data for BTCUSDT, ETHUSDT and SOLUSDT.
- Timeframes: 1m, 3m, 5m, 15m, 30m, 1h, 4h and 1d.
- Chart types: candles, bars, Heikin Ashi, line, area and baseline.
- Pan, pinch zoom, mouse-wheel zoom, date navigation, ranges and real-time scroll.
- Normal, logarithmic, percentage and indexed price scales.
- Symbol comparison overlay.
- Indicators: SMA, EMA, Bollinger Bands, VWAP, Volume, RSI, MACD and ATR.
- Drawing tools: trend line, ray, horizontal/vertical line, rectangle, parallel channel, Fibonacci, measure, long-position and text.
- Drawing persistence, undo/redo, hide, lock, object tree and workspace import/export.
- Alerts that trigger while the page is running.
- Bar replay.
- Screenshot export.
- Chart templates.
- Long-only Galka paper bot for all three symbols with six averaging levels.
- Paper account, fees, approximate maintenance margin, realized/unrealized PnL and CSV export.
- Mobile and desktop layouts.

## Not equivalent to TradingView proprietary products

The following require TradingView Advanced Charts/Trading Platform licensing or substantial additional engineering and are not claimed as complete:

- Pine Script compatibility.
- The full TradingView indicator catalog.
- The full 80–110+ smart drawing-tool catalog.
- Broker routing and real exchange orders.
- Cloud synchronization, social features, news and economic calendars.
- Full DOM/order-book trading, options chains and broker-specific order tickets.
- Server-side 24/7 automation while the phone/browser is closed.

All exchange activity in this repository remains paper-only.
